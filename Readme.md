# 💬 WebSocket Чат на Spring Boot

Простой веб-чат в реальном времени с использованием **WebSocket**, **STOMP** и **SockJS**.  
Позволяет пользователям обмениваться сообщениями без перезагрузки страницы.

## ✨ Возможности

- Мгновенная отправка и получение сообщений
- Поддержка русского языка (UTF-8)
- Автоматическое переподключение при разрыве соединения (SockJS)
- Простой веб-интерфейс
- Работает в нескольких окнах браузера одновременно

## 🛠 Технологии

| Технология | Назначение |
|------------|------------|
| Spring Boot 3.2.4 | Backend-фреймворк |
| Spring WebSocket | Поддержка WebSocket |
| STOMP | Протокол обмена сообщениями |
| SockJS | Fallback-механизм |
| JavaScript + HTML | Клиентская часть |

## 📋 Требования

- Java 17 или выше
- Maven 3.6+ (или использование встроенного Maven Wrapper)
- Браузер с поддержкой WebSocket

## 🚀 Быстрый старт

### 1. Клонирование репозитория

```bash
git clone https://github.com/Sognatore-coder/ChatWebSocket.git
cd websocket-demo
```

## Запуск приложения
### Windows (CMD или PowerShell)
```bash
mvnw.cmd spring-boot:run
```

### macOS / Linux
```bash
./mvnw spring-boot:run
```
## Открыть чат в браузере
Перейдите по адресу: http://localhost:8080
Введите ваше имя
Нажмите Connect
Начинайте общение!
